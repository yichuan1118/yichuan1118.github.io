Setup sublime text 3 to run Java directly

Add a new file JavaC.sublime-build to {sublime config}\Packages\User\ with the following content:

{
    "shell_cmd": "javac \"$file\" && java -cp \"$file_path\" \"$file_base_name\"",
    "file_regex": "^(...*?):([0-9]*):?([0-9]*)",
    "selector": "source.java"
}
